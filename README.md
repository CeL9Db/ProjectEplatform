# Project E-platform

## Overview
Project E-platform is an innovative e-learning platform designed to facilitate online learning experiences through a robust Java backend and a user-friendly web frontend. This platform aims to provide a seamless educational experience for both learners and educators.

## Future Features
- **Payment Integration**: We plan to integrate PayPal for processing payments efficiently and securely. 
- **Code Delivery**: Course-related game codes will be delivered directly to users via email, ensuring easy access to necessary materials.

## Installation Instructions
1. **Clone the repository**:  
   ```bash
   git clone https://github.com/CeL9Db/ProjectEplatform.git
   ```
2. **Navigate to the project directory**:  
   ```bash
   cd ProjectEplatform
   ```
3. **Backend Setup**:  
   - Ensure you have Java 11+ installed.  
   - Build the backend application with Maven:  
     ```bash
     mvn clean install
     ```
4. **Frontend Setup**:  
   - Navigate to the frontend directory:  
     ```bash
     cd frontend
     ```  
   - Install dependencies:  
     ```bash
     npm install
     ```  
   - Start the web server:  
     ```bash
     npm start
     ```

## Features
- **Course Management**: Instructors can create and manage courses easily.
- **Student Progress Tracking**: Learners can track their progress throughout the course.
- **Interactive Learning Materials**: Engage users with interactive elements within courses.
- **User Profiles**: Both learners and instructors can create profiles to manage their courses and learning experiences.

## Project Structure
- **/backend**: Contains the Java-based server code.
- **/frontend**: Contains the web application code.
- **/docs**: Documentation and guides for users and developers.
