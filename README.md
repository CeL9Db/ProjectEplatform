# Project Eplatform

Project Eplatform is a comprehensive game sales platform that provides a wide variety of games for users. While it includes educational games, the platform is not limited to them. We aim to cater to all gamers with diverse interests.

## Upcoming Features

In our next update, we will be integrating PayPal for seamless transactions, making it easier and more secure for users to purchase games. Additionally, we will introduce an email delivery system for game codes, ensuring that users receive their game purchases promptly and efficiently.
